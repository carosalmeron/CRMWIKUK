📥 Importadores de datos

Zona reservada para administradores. Subida mensual de la base de datos.

📋 Orden recomendado de importación: 1️⃣ Primero Clientes (actualiza vendedores y bloqueos) → 2️⃣ Después Ventas (heredan vendedor actual de cada cliente)
👥

Importar Clientes

Carga el Excel mensual de clientes. Detecta nuevos, reasignaciones de vendedor y bloqueos. No borra: los desaparecidos se marcan como fuera de cartera.

Detecta:
  1. 🆕 Clientes nuevos
  2. 🔄 Cambios de vendedor (GRUPOAGENTE)
  3. 🚫 Bloqueos (BLQ=SI)
  4. 📤 Fuera de cartera (no llegan en Excel)
Abrir importador →
📈

Importar Ventas

Carga el Excel mensual de ventas (cliente × artículo × año). Detecta automáticamente clientes perdidos y en riesgo. Hereda el vendedor actual desde la BD de clientes.

Calcula:
  1. 🔴 Perdidos (vendían año ant., 0 ahora)
  2. 🟠 En riesgo (-30% o más)
  3. 🟡 Atención (-10% a -30%)
  4. 👤 Vendedor heredado de la BD de clientes
